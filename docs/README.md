# OpenFSW Documentation

This folder contains design notes and supporting material for the OpenFSW flight software + simulation + ground segment.

## Index

- `docs/validation.md` — how to run the complete pipeline and interpret artifacts
- `docs/simulation.md` — simulation architecture, models, units, and outputs
- `docs/ground.md` — ground-segment packet formats, examples, and tooling
- `docs/firmware.md` — firmware build structure and constraints (bare-metal)
